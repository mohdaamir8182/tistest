# The Instant Solutions Faisalabad Test for React Native Dev

## How to Run?

##### 1): Clone the project
##### 2): Navigate into project root directory
##### 3): Run 'yarn' or 'npm install' - This will install all required libraries and packages.
##### 4): Run the project using 'npx react-native run-android' for ANDROID 
##### 5): Run the project using 'npx react-native run-ios' for IOS

##### DON'T forget to run 'ad ios && pod install' before running for ios.
